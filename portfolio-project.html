Portfolio Site

Simple site using HTML, CSS and JavaScript

I developed my first portfolio site (the one you're on now) using HTML, CSS, and Bootstrap. It was a challenging but rewarding experience. I began by learning the basics of HTML and CSS, which allowed me to structure and style the content on my site. I then discovered Bootstrap, a framework that made it easier for me to create a responsive and visually appealing layout.


Using Bootstrap, I was able to quickly design and implement various elements on my site, such as the navigation bar, hero image, and grid system. I also learned how to customize the look and feel of my site by modifying the default Bootstrap styles with my own CSS code.


Overall, building my first portfolio site was a great opportunity for me to learn and apply new skills in web development. It was a lot of work, but the end result was a professional-looking portfolio that I am proud to share with others.
